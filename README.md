This is a console-based Brick Breaker game developed using Java.
The game runs on a 7x7 grid where the player controls the ball using commands like St, Lt, and Rt.
Bricks break when hit diagonally, and the ball count decreases after each successful hit.
The project follows a modular structure using Main, Service, and Bean classes.
